# kupec

Аналитический дашборд маркета для GTA RP: цены рынка, выгода крафта и перекупа,
кухня, движения цен, журнал сделок, избранное. Работает на публичном API игрового
проекта. Инструмент неофициальный и бренд-нейтральный.

Монорепо: общая логика (`@kupec/core`) переиспользуется вебом (Next.js) и мобилкой
(Expo). Бэкенд — свой Postgres + поллер + API.

## Быстрый старт (локально, с живыми данными)

Нужны: Node ≥20, pnpm (`corepack enable pnpm`), Docker.

```bash
./start-local.command
```

Одна команда: при первом запуске сама поставит зависимости, поднимет базу, API и
веб, откроет `http://localhost:3000`. **Сервер выбери в шапке** (выпадашка RU1–RU19)
и нажми «🔄 С сервера» — цены выбранного сервера подтянутся вживую (сервер → БД → UI).
`Ctrl+C` — остановить всё.

Данные обновляются автоматически каждые ~30 минут (лимит API — 5 запросов/60с
соблюдается). Аккаунт (кнопка «Войти») синхронизирует журнал сделок и избранное.

## Структура

| Пакет | Назначение |
|---|---|
| `packages/core` | вся расчётная логика + справочники, без DOM (тесты: vitest) |
| `packages/client` | общий API-клиент для веба и мобилки |
| `packages/db` | схема Postgres, миграции, запросы |
| `services/poller` | опрос публичного API → БД (рейт-лимит 5/60с) |
| `services/api` | Hono API: рынок, аккаунты, журнал |
| `apps/web` | веб-дашборд (Next.js 15 + React 19 + Tailwind) |
| `apps/mobile` | мобильное приложение (Expo + React Native) |

Команды: `pnpm test`, `pnpm typecheck`, `pnpm --filter @kupec/web dev`.
Деплой на сервер — см. [`docs/DEPLOY.md`](docs/DEPLOY.md).

## Статус

Рабочий прототип — `majestic_market_dashboard.html` (референс). Прод-версия
(это монорепо) готова к локальному использованию; публичный запуск/монетизация —
после согласования условий с игровым проектом.
